# FreeRADIUS授权
这部分可以看作是我们迄今为止所涵盖的主题的概述，也是一个复习者，然后再进行授权的更多动手练习。

+ 请求从NAS（客户端）发送到FreeRADIUS（服务器）。
+ 这些请求由虚拟服务器处理，虚拟服务器在FreeRADIUS配置中定义。默认虚拟服务器称为默认。
+ 处理传入请求的方式取决于虚拟服务器文件中各个部分的配置。
+ 请求本身由逻辑顺序中的各个部分处理。授权部分始终在authenticate部分之前处理Access-Request数据包。 preacct部分同样总是在accounting部分之前处理Accounting-Request数据包。
+ 虽然部分顺序无法更改，但我们在部分内部有很多灵活性来操作请求。

下一部分将是处理请求的基本介绍。通过使用比较和逻辑，我们可以控制请求的流程并操纵属性。
